# Gemini CLI App (Chat + Summarizer)

A simple Python command-line app that uses the Google Gemini API to:
- Send a text prompt and display the model's response (Chat mode)
- Summarize long text into a short, easy-to-understand summary (Summarization mode)

This project is designed for learning/assignment purposes. The code is synchronous, beginner-friendly, and uses the official `google-genai` library with the `gemini-1.5-flash` model.

## Requirements
- Python 3.x
- A Google Gemini API key (get one at https://aistudio.google.com/apikey)

## Setup
1. (Optional) Create and activate a virtual environment
   - Windows PowerShell:
     ```powershell
     cd gemini_cli_app
     python -m venv .venv
     .\.venv\Scripts\Activate.ps1
     ```
   - macOS/Linux (bash/zsh):
     ```bash
     cd gemini_cli_app
     python3 -m venv .venv
     source .venv/bin/activate
     ```

2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

3. Add your API key
   - Open `config.py` and replace the placeholder value:
     ```python
     # Replace "API_KEY" with your actual Gemini API key from Google AI Studio.
     API_KEY = "YOUR_REAL_KEY"
     ```

4. Run the app
   ```bash
   python main.py
   ```

## How to Use
- You will see a simple menu:
  ```
  ====================================
          Gemini CLI Application
  ====================================
  1. Chat mode (ask a question)
  2. Summarization mode
  3. Exit
  ====================================
  ```

- Option 1: Chat mode
  - Enter your question or prompt and press Enter.
  - The app prints the model's reply.

- Option 2: Summarization mode
  - Paste or type multiple lines of text.
  - Press Enter on a blank line to finish input.
  - The app prints a short summary generated by Gemini.

- Option 3: Exit
  - Cleanly exits the program.
